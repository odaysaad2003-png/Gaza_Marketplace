# Mini Marketplace Gaza

Arabic RTL mini marketplace built with Next.js, TypeScript, Tailwind CSS, shadcn/ui, React Query, Zustand, Zod, and Recharts.

## Features

- Public marketplace homepage
- Products listing
- Product details page
- Create product form with validation
- Favorites system with local storage
- Admin Dashboard
- Admin Products Management
- Admin Analytics Dashboard with Recharts
- Admin Settings UI
- Dark mode
- Responsive RTL UI
- Mock API architecture

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query
- Zustand
- Zod
- Recharts
- Lucide React

```bash
npm install
npm run dev
